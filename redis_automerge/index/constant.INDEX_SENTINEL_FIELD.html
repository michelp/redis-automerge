Sentinel field embedded in every shadow document this module owns. The value is the source Automerge document key, giving the shadow a clear owner stamp. Before overwriting or deleting a shadow at am:idx:<key>, the indexer probes for this field and refuses to touch keys that lack it, so a user’s pre-existing key at e.g. am:idx:foo is never silently clobbered. See SECURITY_AUDIT.md #13.
