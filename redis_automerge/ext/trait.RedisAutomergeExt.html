Expand description

Convenience methods for integrating Automerge with Redis persistence layers.

AOF replication is not handled here. Each AM.* write command in lib.rs calls ctx.replicate(cmd, args) via finalize_write_meta, which causes Redis to append the original command to the AOF file; on restart, the command is replayed by its normal handler. See audit #9 in SECURITY_AUDIT.md for the analysis of why a per-client change buffer was unnecessary.
